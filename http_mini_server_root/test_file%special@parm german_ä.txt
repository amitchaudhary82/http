This is a test file with special characters in the file name.
When using HTTP / URL, special characters must be encoded.


Original file name

test_file%special@parm german_ä.txt



Encoding needed

char    encode?   encoding   reason
------  --------  --------   -------------------------
%       yes       %25        % is escape character
@       safer     %40        reserved character
space   yes       %20        not allowed in URL
ä       yes       %C3%A4     UTF-8 encoding
_       no        _          safe character
.       no        .          safe character



Important

Do NOT use "/" in file name.
"/" is treated as folder separator in URL.



Encoded file name to use in URL

/test_file%25special%40parm%20german_%C3%A4.txt



Rule

Special characters must be percent-encoded.
Non-ASCII characters must use UTF-8,
then each byte becomes %HH.
